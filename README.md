# Crafty
This is a Java (Spring) + React application/site for sharing handcrafted items.

Technologies:
- Java 8+
  - Maven 3.5+
- Node 6+
  - React 16+
- MySQL server

Setup (after java, mvn, node are installed):
- Java
  - mvn spring-boot:run
- Node (new terminal)
  - cd crafty-app
  - npm install
  - npm start
- MySQL - start on :3306 (default)
  - run setup.sql
- Browser
  - localhost:3000
